# Elwazi Schema.org Metadata Catalog

This repository hosts Pathfinder site codebooks from the Elwazi catalog together with their translated Schema.org JSON-LD metadata records.

## Pathfinder sites
- AHRI
- DGH
- IRESSEF

## Purpose
The purpose of this work is to improve the machine readability, interoperability, and reuse of metadata by translating spreadsheet-based codebooks into structured Schema.org JSON-LD.

## Contents
Each Pathfinder site folder contains:
- original codebooks
- translated JSON-LD files
- optional mapping notes or supporting documentation
