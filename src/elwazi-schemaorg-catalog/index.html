Elwazi Schema.org Metadata Catalog

This site provides access to Pathfinder site codebooks hosted through the Elwazi catalog and their translated Schema.org JSON-LD records for improved machine readability, interoperability, and FAIR-aligned metadata reuse.

Purpose: The goal of this catalog is to support translation of metadata from spreadsheet-based codebooks into structured Schema.org JSON-LD that can be reused across different dataset types and discovery workflows.

What is included

  • Original spreadsheet codebooks from the Elwazi catalog
  • Translated Schema.org JSON-LD metadata files
  • Site-specific notes on mapping and metadata translation where available

Review focus

  • Alignment with Schema.org and Science-on-Schema.org guidance
  • Improved machine readability of codebook metadata
  • Consistency across Pathfinder sites
  • Support for reusable metadata patterns across tabular and geospatial datasets
